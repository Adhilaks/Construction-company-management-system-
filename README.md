# Magizh Construction Management App

A simple Tkinter-based GUI application for managing customers and displaying services for Magizh Construction, specializing in waterproofing and construction services.

## Features

- Customer management (add and view customers)
- Service display with images
- Contact information and WhatsApp integration
- SQLite database for data storage

## Installation

1. Ensure you have Python 3.x installed.
2. Install the required dependencies:

   ```
   pip install -r requirements.txt
   ```

## Usage

Run the application by executing:

```
python main.py
```

The app will open a GUI window with options to navigate through Home, About, Services, and Contact pages.

## Dependencies

- Pillow (for image handling, optional but recommended for full functionality)
- Tkinter (included with Python)
- SQLite3 (included with Python)

## Database

The app uses a local SQLite database (`magizh.db`) to store customer information.
